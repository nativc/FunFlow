FunFlow 是一个短视频浏览与分享应用，致力于为用户提供流畅、有趣的短视频消费体验。

## Features

- **视频推送**：兴趣推送、相似视频推荐、标签列表浏览、热门视频排行榜
- **视频互动**：视频点赞、收藏、评论
- **用户账号**：登录、注册、个人信息展示与修改
- **视频创作**：本地视频上传
- **个人空间**：关注/粉丝列表、作品展示、收藏夹、喜欢列表、浏览历史
